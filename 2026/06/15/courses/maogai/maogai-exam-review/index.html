毛泽东思想和中国特色社会主义理论体系概论 期末考点速记

本文基于 北邮近两年毛概期末真题（2024–2025 学年第二学期、2023–2024 学年第二学期，均含完整答案）逆向整理，覆盖第一章～第七章（毛泽东思想 → 科学发展观）。 闭卷背诵导向：先看 🎯 老师最后一课划重点（哪章出论述、主观题怎么答），再看 §零 考情 与 §一 真题考点分布，按章背 §二～§八，考前突击只看 §十一 时间线速记表 即可。

⚠️ 本课程不含「习近平新时代中国特色社会主义思想」——那是单独的《习概》课。两套真题的最后一题都止于科学发展观，不要多背。

零、考情速览：先搞清楚考什么

题型 题量 × 分值 合计 难度与策略
一、单选题 20 × 1 20 分 全是「时间 / 会议 / 著作 / 人物 / 概念」对应，靠记忆，时间线表是命根子
二、简答题 4 × 10 40 分 真题标注「只需回答是什么」——背定义和要点条目即可，不必展开
三、论述题 2 × 20 40 分 材料题（给一段讲话/文件），要求「系统地论述」，分点 + 关系，材料本身不给分
合计 — 100 分 闭卷

三条备考结论（来自真题规律）：

  1. 得简答论述者得天下：简答+论述占 80 分，且答案高度固定、可预测。把 §九 简答题 和 §十 论述题 背熟，及格无忧。
  2. 单选靠时间线：20 道单选几乎都能在 §十一 速记表 里找到。重点记「第一次提出 / 标志 / 核心 / 精髓 / 灵魂 / 本质 / 根本任务 / 第一要义」这类「定性词」。
  3. 两年高度重复：活的灵魂、三大法宝、农村包围城市三关系、一化三改、论十大关系、三个代表、科学发展观——连续两年都考，必背。

二、毛泽东思想及其历史地位（第一章）

2.1 形成与发展过程（论述高频，23-24-2 论述②原题）

记忆主线：萌芽→形成→成熟→继续发展，每个阶段对应一个「时期 + 标志性理论」。

阶段 时期 标志
萌芽 党的创立和大革命时期 对中国社会和革命基本问题的初步探索
初步形成 土地革命战争前中期 提出「农村包围城市、武装夺取政权」，开辟井冈山道路
走向成熟 遵义会议后到抗日战争时期 系统阐述新民主主义革命理论，实现马克思主义中国化第一次历史性飞跃
继续发展 解放战争和新中国成立后 人民民主专政、社会主义改造理论、《论十大关系》《正确处理人民内部矛盾》

2.2 活的灵魂：三个基本方面（两年必考单选，必背）

实事求是、群众路线、独立自主。（口诀：实·群·独）

  • 实事求是：思想路线的核心，毛泽东思想的精髓。
  • 群众路线：一切为了群众、一切依靠群众，从群众中来、到群众中去。
  • 独立自主：自力更生，把立足点放在自己力量的基点上。

2.3 历史地位与主要著作（单选点）

  • 指导地位确立：1945 年中共七大把毛泽东思想确立为党的指导思想并写入党章。
  • 「马克思主义中国化」命题提出：1938 年六届六中全会，毛泽东作《论新阶段》报告首次提出。
  • 毛泽东思想是马克思主义中国化的第一次历史性飞跃的理论成果，是党和人民宝贵的精神财富。

三、新民主主义革命理论（第二章）

全书考查最密集的一章。核心记忆框架：总路线 + 三大对象 + 四大动力 + 三大纲领 + 道路 + 三大法宝。

3.1 新民主主义革命的总路线（24-25-2 简答①）

无产阶级领导的，人民大众的，反对帝国主义、封建主义和官僚资本主义的革命。

拆解记忆：

  • 领导力量：无产阶级（这是新旧民主主义革命的根本区别）。
  • 革命对象（三大敌人）：帝国主义、封建主义、官僚资本主义。
  • 革命动力：无产阶级、农民阶级、城市小资产阶级、民族资产阶级。其中农民阶级是中国革命最基本的动力，无产阶级是领导力量。
  • 革命性质：资产阶级民主主义革命（新式的、特殊的）。
  • 革命前途：社会主义（不是资本主义）。

3.2 三大基本纲领（政治·经济·文化）

纲领 内容要点
政治纲领 推翻帝国主义和封建主义统治，建立无产阶级领导的、以工农联盟为基础的、各革命阶级联合专政的新民主主义共和国
经济纲领 ① 没收封建地主土地归农民所有；② 没收官僚资本归新民主主义国家所有；③ 保护民族工商业
文化纲领（23-24-2 简答①） 民族的、科学的、大众的文化，即无产阶级领导的人民大众的反帝反封建的文化

3.3 农村包围城市、武装夺取政权（两年必考单选）

  • 根本在于处理好三者关系：土地革命、武装斗争、农村革命根据地建设。
    • 土地革命是基本内容，武装斗争是主要形式，农村根据地是战略阵地。
  • 这条道路理论是马克思主义中国化的重要成果，是「工农武装割据」思想的发展。

3.4 三大法宝（24-25-2 论述①、23-24-2 单选）

统一战线、武装斗争、党的建设。出自 1939 年《〈共产党人〉发刊词》。

  • 统一战线：包含两个联盟——工农联盟（基础、主要）+ 与非劳动人民的联盟。
  • 武装斗争：中国革命的主要形式；坚持党对军队的绝对领导是建军根本原则。
  • 党的建设：「伟大的工程」；形成理论联系实际、密切联系群众、批评与自我批评三大优良作风。
  • 三者关系：统一战线和武装斗争是战胜敌人的两个基本武器，党的组织是掌握这两个武器、向敌冲锋陷阵的英勇战士。

四、社会主义改造理论（第三章）

4.1 过渡时期总路线：一化三改（24-25-2 单选、23-24-2 简答②）

党在过渡时期总路线 = 「一化三改」，又称「一体两翼」。

  • 一化（主体）：逐步实现国家的社会主义工业化。
  • 三改（两翼）：逐步实现对个体农业、个体手工业、资本主义工商业的社会主义改造。
  • 实质：使生产资料的社会主义公有制成为我国社会唯一的经济基础。

4.2 社会主义改造的道路与经验

改造对象 方针 / 形式
农业、手工业 方针「积极引导、稳步前进」，原则「自愿互利、典型示范、国家帮助」；步骤：互助组 → 初级社 → 高级社
资本主义工商业 和平赎买、国家资本主义，由低级到高级（加工订货 → 公私合营 → 全行业公私合营）
  • 新民主主义社会的三种主要经济成分：社会主义经济、个体经济、资本主义经济（24-25-2 单选）。
  • 意义：1956 年三大改造基本完成，社会主义基本制度在中国确立，是中国历史上最深刻的社会变革。

五、社会主义建设道路初步探索的理论成果（第四章）

5.1 《论十大关系》（1956，两年必考单选）

  • 标志：党探索中国社会主义建设道路的良好开端。
  • 基本方针：调动一切积极因素为社会主义事业服务。

5.2 《关于正确处理人民内部矛盾的问题》（1957）

  • 提出社会主义社会两类不同性质的矛盾：敌我矛盾、人民内部矛盾。
  • 社会主义社会的基本矛盾仍是生产力和生产关系、经济基础和上层建筑的矛盾（非对抗性，可经制度自身调整解决）。

5.3 其他单选点

  • 四个现代化构想：1954 年 9 月，周恩来在一届全国人大一次会议政府工作报告中第一次提出。
  • 第一届全国人民代表大会召开时间：1954 年 9 月（通过第一部《中华人民共和国宪法》）。

六、邓小平理论（第五章）

6.1 核心定性词（单选高频，连背）

问题 答案
邓小平理论的精髓 解放思想、实事求是
首次明确「建设有中国特色的社会主义」 1982 年党的十二大开幕词
系统论述社会主义初级阶段理论的会议 党的十三大
正式确立社会主义市场经济体制改革目标的会议 党的十四大（1992）
社会主义的根本任务 发展生产力
社会主义社会发展的直接动力 改革
中国解决所有问题的关键 发展（解放和发展生产力）

6.2 社会主义本质（邓小平，必背）

解放生产力，发展生产力，消灭剥削，消除两极分化，最终达到共同富裕。

6.3 社会主义市场经济理论的内涵（24-25-2 简答②）

  1. 计划经济和市场经济不是社会制度的标志——计划经济不等于社会主义，市场经济不等于资本主义；
  2. 计划和市场都是经济手段，各有优势，社会主义搞市场经济要把两者结合；
  3. 市场经济作为资源配置方式本身不具有制度属性，可与不同社会制度结合。坚持社会主义制度与市场经济结合，是社会主义市场经济的特色与优势所在。

6.4 「三个有利于」标准（23-24-2 简答③）

判断改革开放各项工作成败得失，看： ① 是否有利于发展社会主义社会的生产力； ② 是否有利于增强社会主义国家的综合国力； ③ 是否有利于提高人民的生活水平。

6.5 邓小平理论的历史地位（24-25-2 简答③）

  1. 马克思列宁主义、毛泽东思想的继承和发展；
  2. 中国特色社会主义理论体系的开篇之作；
  3. 改革开放和社会主义现代化建设的科学指南。

七、「三个代表」重要思想（第六章）

7.1 核心观点（24-25-2 论述②，必背）

始终代表中国先进生产力的发展要求、中国先进文化的前进方向、中国最广大人民的根本利益。

  • 回答的基本问题：建设什么样的党、怎样建设党（23-24-2 单选）。
  • 本质：立党为公、执政为民。
  • 集中概括：始终代表（生产力·文化·人民利益）。
  • 贯彻的核心在坚持党的先进性（24-25-2 单选）；关键在坚持与时俱进。

7.2 其他单选点

  • 发展是党执政兴国的第一要务。
  • 加强社会主义思想道德建设是发展先进文化的重要内容和中心环节。
  • 建设社会主义政治文明，最根本是坚持党的领导、人民当家作主和依法治国的有机统一。
  • 「全面建设小康社会」是实现现代化第三步战略目标承上启下的发展阶段。

八、科学发展观（第七章）

8.1 科学内涵（四句话，必背；23-24-2 简答④）

维度 内容
第一要义 发展
核心立场 以人为本
基本要求 全面协调可持续
根本方法 统筹兼顾

口诀：发展为要、以人为本、全面协调、统筹兼顾。 24-25-2 单选考「核心立场＝以人为本」；23-24-2 单选考「第一要义」。两个别混！

8.2 单选点

  • 首次提出：2003 年《中共中央关于完善社会主义市场经济体制若干问题的决定》。
  • 主攻方向：加快转变经济发展方式，把经济结构战略性调整作为主攻方向。
  • 社会和谐是中国特色社会主义的本质属性。

8.3 构建社会主义和谐社会的总要求（24-25-2 简答④）

民主法治、公平正义、诚信友爱、充满活力、安定有序、人与自然和谐相处。 （口诀：民·公·诚·活·安·和，六个词）

九、高频简答题汇总（背诵版）

真题明确：简答「只需回答是什么」。下列答案直接背要点条目即可。

  1. 新民主主义革命总路线 → 无产阶级领导的、人民大众的、反对帝国主义封建主义官僚资本主义的革命。
  2. 新民主主义文化纲领 → 民族的、科学的、大众的文化（无产阶级领导的人民大众的反帝反封建文化）。
  3. 党在过渡时期的总路线 → 「一化三改」：一化＝社会主义工业化（主体）；三改＝对农业、手工业、资本主义工商业的社会主义改造。
  4. 社会主义市场经济理论的内涵 → 见 §6.3（三条）。
  5. 邓小平理论的历史地位 → 见 §6.5（继承发展 / 开篇之作 / 科学指南）。
  6. 改革成败得失的衡量标准 → 三个有利于（生产力 / 综合国力 / 人民生活水平）。
  7. 科学发展观的科学内涵 → 以人为本、全面发展、协调发展、可持续发展、统筹兼顾。
  8. 构建社会主义和谐社会的总要求 → 民主法治、公平正义、诚信友爱、充满活力、安定有序、人与自然和谐相处。

10.8 论述题通用答题法（四步拿分法 + 评分逻辑）

先懂评分逻辑：论述题踩点给分——阅卷看三样东西：① 写出多少个"采分点"（课本要点）；② 逻辑/层次是否清楚；③ 字数是否够。材料本身不给分。所以总策略只有一句：多分点、写够课本要点、每点带小标题。一道 20 分题争取写满 4~6 个采分点、不少于 300~400 字。

四步拿分法：

步骤 做什么 要点
一·审题 读材料找"题眼"，判断考哪一章、哪个理论、哪类问法 圈出材料里的人名/会议/著作/关键词，对应到课本；判断问的是"经验 / 内涵 / 意义 / 过程 / 关系"哪一类
二·点题 开头一句话点出"是什么"（定义 / 总路线 / 总要求） 先给标准定义或总括句亮明观点，先拿"概念分"
三·分点展开 按 ①②③ 分点，每点"是什么 + 为什么（地位/作用/要求）" 每点起一个加粗小标题；点数宁多勿少；史实 + 理论结合
四·收尾升华 写"三者关系 / 历史意义 / 现实意义" 关系类题写内在联系；其余写意义、地位、对今天的启示

两类题的固定结构（覆盖几乎所有真题）：

  • 「经验 / 内容 / 内涵」类（三大法宝、三个代表、科学发展观内涵）：点题 → 逐条展开（每条含义 + 作用）→ 关系 / 意义收尾。
  • 「过程 / 发展」类（毛泽东思想形成发展、基本路线发展完善）：按阶段/时间分段，每段"时期 + 标志性成果"，结尾点历史地位。

动笔前先在草稿列点：定义点 → 内容点（分条）→ 地位/意义点 → 关系/现实点，列够 4~6 点再正式写。

10.9 论述题语言·句式·升华模板库

没思路、凑字数、求稳妥时直接套用；务必用规范政治术语，别写口语。

开头点题句：

  • "……的科学内涵（基本经验 / 核心要义）可以概括为以下几个方面：……"
  • "要正确理解这一问题，必须从……（历史背景 / 理论逻辑）出发，结合……的实践来认识。"

分点过渡句：

  • "第一，……。这是因为……，集中体现了……。"
  • "其次 / 再次，从……方面看，……，这就要求……。"

收尾升华句（万能保底，三选一）：

  • 关系类："三者相互联系、相互促进，是一个有机统一的整体，统一于……的实践。"
  • 意义类："这一理论（经验）是马克思主义中国化的重要成果，不仅指导了当时的革命 / 建设实践，对今天坚持和发展中国特色社会主义仍具有重要的指导意义。"
  • 地位类："它是对……的继承和发展，是……理论体系的重要组成部分，是党必须长期坚持的指导思想。"

规范术语清单：科学内涵、历史地位、根本任务、指导思想、有机统一、与时俱进、解放和发展生产力、实事求是。时间 / 会议 / 著作务必写准（见 §十一 时间线）。

十二、易混易错辨析

  • 毛泽东思想活的灵魂 ≠ 三大法宝：灵魂是「实事求是·群众路线·独立自主」（思想方法）；法宝是「统一战线·武装斗争·党的建设」（革命经验）。两年都拿这两个互设干扰项。
  • 精髓 vs 核心 vs 第一要义：精髓＝实事求是（思想路线）；三个代表核心在党的先进性；科学发展观第一要义＝发展、核心立场＝以人为本。
  • 新民主主义革命的前途是社会主义，不是新民主主义/资本主义。
  • 革命最基本动力是农民阶级，领导力量是无产阶级——问法不同答案不同。
  • 十二大 vs 十三大 vs 十四大：十二大提「中国特色社会主义」；十三大讲「初级阶段理论」；十四大定「市场经济体制」目标。
  • 「四个现代化」首次提出是 1954 年周恩来（不是 1964 / 1975 那次重申，更不是 1978）。

一、单选题（每题 1 分，共 20 分）参考答案

  1. 毛泽东思想初步形成于 → 土地革命战争时期
  2. 把毛泽东思想确立为党长期坚持的指导思想 → 党的七大（1945）
  3. 1927 年江西永新「支部建在连上」 → 三湾改编
  4. 毛泽东思想活的灵魂 → 实事求是、群众路线、独立自主
  5. 1939《中国革命和中国共产党》首次提出 → 「新民主主义的革命」
  6. 农村包围城市，处理好三者关系 → 土地革命、武装斗争、农村革命根据地建设
  7. 新民主主义社会三种经济成分 → 社会主义经济、个体经济和资本主义经济
  8. 「一化三改」的「一化」 → 社会主义工业化
  9. 《论十大关系》标志 → 党探索中国社会主义建设道路的良好开端
  10. 周恩来首提「四个现代化」构想 → 1954 年 9 月
  11. 邓小平「走自己的路，建设有中国特色的社会主义」 → 党的十二大
  12. 改革两条根本原则 → 一是公有制为主体，一是共同富裕
  13. 中国解决所有问题的关键 → 解放和发展生产力
  14. 贯彻「三个代表」核心 → 党的先进性
  15. 承上启下、完善市场经济体制的关键阶段 → 全面建设小康社会
  16. 政治文明最根本 → 党的领导、人民当家作主和依法治国的有机统一
  17. 2003 年首次提出科学发展观的文件 → 《中共中央关于完善社会主义市场经济体制若干问题的决定》

  19. 科学发展观的核心立场 → 以人为本
  20. 中国特色社会主义的本质属性 → 社会和谐

二、简答题（每题 10 分）

  1. 新民主主义革命的总路线 → 无产阶级领导的、人民大众的、反对帝国主义、封建主义、官僚资本主义的革命。
  2. 社会主义市场经济理论的内涵 → 见 §6.3。
  3. 邓小平理论的历史地位 → ① 马列、毛泽东思想的继承和发展；② 中国特色社会主义理论体系的开篇之作；③ 改革开放和社会主义现代化建设的科学指南。
  4. 构建社会主义和谐社会的总要求 → 民主法治、公平正义、诚信友爱、充满活力、安定有序、人与自然和谐相处。

二、简答题（每题 10 分）

  1. 新民主主义的文化纲领 → 民族的、科学的、大众的文化。
  2. 党在过渡时期的总路线 → 「一化三改」：一化＝社会主义工业化（主体）；三改＝对个体农业、手工业、资本主义工商业的社会主义改造。
