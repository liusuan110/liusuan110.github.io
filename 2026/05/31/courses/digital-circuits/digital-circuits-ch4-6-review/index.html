数字电路与逻辑设计 4-6章知识点与做题方法

这篇只整理第 4-6 章：触发器、时序逻辑电路、中规模时序集成电路。重点不是把课件重抄一遍，而是把考试做题时真正会用到的公式、状态表方法和常见套路集中放在一起。

零、4-6 章主线

这三章可以按一条线理解：

触发器→状态方程→状态表/状态图→计数器与序列发生器→中规模芯片应用
  • 第 4 章：集成触发器。解决“一个存储单元如何记 0/1、什么时候改变状态、下一状态怎么由输入决定”。
  • 第 5 章：时序逻辑电路。解决“多个触发器加组合逻辑后，如何分析状态变化，如何设计指定功能电路”。
  • 第 6 章：中规模时序集成电路及应用。解决“已有计数器、移位寄存器芯片时，如何接成任意进制计数器、序列发生器、分频器等”。

看到题目时先判断它属于哪一类：

  1. 给电路让你分析：写方程，列状态表，画状态图，判断功能。
  2. 给功能让你设计：定状态，选触发器，求激励方程，检查自启动。
  3. 给芯片让你接线：看清清零、置数、使能、进位端的有效电平和同步/异步性质。

一、第 4 章：集成触发器

1.1 时序电路和组合电路的区别

组合逻辑电路的输出只由当前输入决定：

Y=f(X)

时序逻辑电路的输出和下一状态不仅与当前输入有关，还与原状态有关：

Qn+1=f(Qn,X)

其中：

  • Qn ：触发器现态。
  • Qn+1 ：触发器次态。
  • X ：外部输入。

所以时序题的核心就是：现态 + 输入 + 触发条件 = 次态。

1.2 基本触发器公式

常见触发器的特性方程如下：

触发器 功能 特性方程 重点
RS 触发器 置 1、置 0、保持 Qn+1=S+R―Qn 主动高输入时约束 RS=0
D 触发器 锁存输入 D Qn+1=D 设计题最简单
JK 触发器 保持、置 1、置 0、翻转 Qn+1=JQn―+K―Qn 无禁态，计数器常用
T 触发器 T=1 翻转， T=0 保持 Qn+1=TQn―+T―Qn 等价于 Qn+1=Qn⊕T
T′ 触发器 每来一次有效时钟就翻转 Qn+1=Qn― 分频、计数常用

RS 触发器要特别注意有效电平：

  • 与非门构成的基本 RS 触发器通常是低电平有效输入，常写作 S― 、 R― 。
  • 或非门构成的基本 RS 触发器通常是高电平有效输入。
  • 做题时不要只背公式，先看图中输入端有没有小圆圈。

1.3 触发方式

题目经常混合考“触发器类型”和“触发方式”，这两个要分开看。

触发方式 状态改变时刻 做波形题时怎么看
电平触发 CP 为有效电平期间都可能变化 在有效电平整段内跟踪输入
主从触发 通常在 CP 下降沿输出改变 注意“一次翻转”现象
边沿触发 只在有效边沿附近改变 只取边沿到来前瞬间的输入值

边沿触发器最适合做同步时序电路，因为各触发器可以在同一时钟边沿同时改变状态。

1.4 建立时间和保持时间

边沿触发器不是只看一个理想瞬间，输入信号必须满足时间要求：

  • 建立时间 tsu ：输入信号要在有效边沿到来前提前稳定。
  • 保持时间 th ：输入信号要在有效边沿之后继续保持一小段时间。

若不满足，触发器可能进入不确定状态。考试中如果问“为什么电路工作不可靠”，常从建立时间、保持时间、竞争冒险这些角度答。

1.5 波形题做法

画触发器波形时按这个顺序来：

  1. 先看异步置位、异步清零端。若有效，它们优先决定 Q ，时钟可以先不看。
  2. 看触发器类型：RS、D、JK、T 还是 T′ 。
  3. 看时钟触发方式：高电平、低电平、上升沿、下降沿。
  4. 到每一个有效触发时刻，代入特性方程求 Qn+1 。
  5. 标清初态。如果题目没给初态，一般不能唯一确定完整波形。

最容易错的地方：

  • JK 触发器 J=K=1 时是翻转，不是不确定。
  • RS 触发器禁态要看有效电平。
  • 主从 JK 触发器输出通常在下降沿变，但输入在 CP 有效期间的变化可能影响主触发器。
  • 边沿触发 D 触发器只看边沿前的 D ，不要把边沿后的变化算进去。

二、第 5 章：时序逻辑电路分析

2.1 三类方程

分析时序逻辑电路，最重要的是把电路图翻译成方程。

激励方程，也叫驱动方程：

Yi=fi(X,Qn)

这里 Yi 是送到触发器输入端的信号，比如 Di 、 Ji 、 Ki 、 Ti 。

状态方程：

Qin+1=hi(X,Qn)

它说明每个触发器的次态如何由现态和输入决定。

输出方程：

Zi=gi(X,Qn)

输出方程用来判断电路最终功能。

2.2 Moore 型和 Mealy 型

类型 输出依赖 特点
Moore 型 只依赖状态 Qn 输出稳定，状态图中输出常标在状态旁
Mealy 型 依赖状态 Qn 和输入 X 反应快，状态图中输出常标在转移边上

如果题目问“输出是否与输入直接有关”，就是在区分 Moore 和 Mealy。

2.3 同步时序电路分析步骤

同步时序电路中，所有触发器共用同一个时钟，因此可以按同一拍同时更新。

标准步骤：

  1. 从组合逻辑写出各触发器的激励方程。
  2. 写出输出方程。
  3. 把激励方程代入触发器特性方程，得到状态方程。
  4. 枚举 Qn 和输入 X ，列状态转换表。
  5. 根据状态表画状态图。
  6. 根据有效循环判断电路功能，如计数器、序列检测器、序列发生器等。

对于 D 触发器，这一步特别简单：

Qin+1=Di

对于 JK 触发器，需要代入：

Qin+1=JiQin―+Ki―Qin

2.4 异步时序电路分析步骤

异步时序电路中，各触发器的时钟不完全相同，有的触发器输出会成为后级触发器的时钟。

分析时要注意：

  • 不是所有触发器在同一时刻改变。
  • 只有收到有效触发边沿的触发器才更新。
  • 状态变化可能一级一级传播。

做异步计数器题时，不要直接把所有触发器一起代入次态。更稳的方法是：

  1. 找第一级触发器的时钟。
  2. 看第一级翻转后，是否给第二级产生有效边沿。
  3. 如果产生，再更新第二级。
  4. 依次往后传播，得到最终状态。

2.5 寄存器、移位寄存器和计数器

寄存器用于存储多位二进制信息。 n 位寄存器一般需要 n 个触发器。

移位寄存器既能存储，也能在时钟作用下移动数据：

  • 左移：二进制数通常相当于乘 2。
  • 右移：二进制数通常相当于除 2。
  • 可以实现串行输入、串行输出、并行输入、并行输出。

计数器用于记录输入脉冲个数。若计数循环中有 M 个有效状态，则称为 M 进制计数器，模为：

M=有效循环状态数

设计 M 进制计数器时，触发器个数 k 满足：

M≤2k

多余状态数为：

2k−M

这些多余状态必须检查能否回到有效循环，也就是自启动问题。

2.6 自启动判断

自启动是计数器和时序逻辑设计里的高频检查点。

判断方法：

  1. 画出完整状态图，不只画有效状态。
  2. 看每一个无效状态下一步会去哪里。
  3. 如果所有无效状态最终都能进入有效循环，则电路能自启动。
  4. 如果某些无效状态形成单独小循环，或停在无效状态，则不能自启动。

设计题里如果卡诺图把无效状态当任意项使用，最后一定要回头检查自启动。

三、第 5 章：时序逻辑电路设计

3.1 一般设计流程

给功能要求设计时序电路，常用流程是：

  1. 分析题意，确定输入、输出和状态含义。
  2. 画原始状态图或列原始状态表。
  3. 状态化简和状态编码。
  4. 选择触发器类型。
  5. 根据状态表求触发器激励方程。
  6. 求输出方程。
  7. 画逻辑电路图。
  8. 检查完整状态图，判断能否自启动。

如果题目是计数器，状态含义通常已经很明确，可以直接从计数序列写状态转换表。

3.2 D 触发器设计

D 触发器设计最直接，因为：

Di=Qin+1

所以只要状态表中已经知道 Qin+1 ，就可以直接用卡诺图化简得到 Di 。

适用场景：

  • 题目没有指定触发器类型。
  • 想减少推导量。
  • 状态转换比较复杂。

3.3 JK 触发器设计

JK 触发器常用于计数器。可以用激励表从现态和次态反推 J 、 K 。

Qn Qn+1 J K
0 0 0 X
0 1 1 X
1 0 X 1
1 1 X 0

其中 X 表示任意项。做题方法：

  1. 列状态转换表。
  2. 对每一位触发器，根据 Qin→Qin+1 查 JK 激励表。
  3. 分别画 Ji 、 Ki 的卡诺图。
  4. 化简得到激励方程。

不要把 JK 的特性方程和激励表混着用。分析题用特性方程，设计题常用激励表更快。

3.4 同步计数器设计

同步计数器所有触发器共用同一 CP。

设计步骤：

  1. 根据模数 M 确定触发器个数 k ：

    M≤2k

  2. 写出计数循环，比如 000→001→010→⋯ 。

  3. 列状态转换表。

  4. 根据所选触发器求激励方程。

  5. 化简组合逻辑。

  6. 检查无效状态能否进入有效循环。

例如设计 5 进制计数器，需要：

5≤23

所以至少用 3 个触发器，有效状态可以取 000,001,010,011,100 ，剩余 101,110,111 是无效状态。

3.5 序列发生器

序列发生器常见两种做法。

计数器型序列发生器

用计数器产生循环状态，再用组合逻辑把每个状态映射成输出序列。适合序列长度和计数模数对应清楚的题。

做法：

  1. 设序列长度为 M 。
  2. 设计 M 进制计数器。
  3. 令每个状态对应一个输出位。
  4. 用卡诺图化简输出函数 Z 。

移位寄存器型序列发生器

用移位寄存器保存最近若干位，反馈逻辑产生新输入。

若序列长度为 M ，一般先选最小 k 满足：

2k−1<M≤2k

然后用 k 位状态表示连续 k 个输出位。若出现重复状态，说明 k 不够或编码方式需要调整。

最长线性反馈移位寄存器序列，也就是 m 序列，长度为：

M=2k−1

注意 m 序列通常不包含全 0 状态，因为全 0 经过异或反馈仍然是全 0。若题目要求自启动，需要额外设计校正逻辑。

四、第 6 章：中规模时序集成电路及应用

第 6 章的核心不是从门级重新设计，而是使用已有的中规模计数器、移位寄存器芯片完成指定功能。

4.1 看芯片题先看端口

遇到计数器芯片或移位寄存器芯片，先把端口性质标出来：

  • 清零端是同步还是异步。
  • 清零端是高有效还是低有效。
  • 置数端是同步还是异步。
  • 置数端是高有效还是低有效。
  • 使能端有几个，是否需要同时有效。
  • 进位端或借位端什么时候有效。
  • 芯片是加计数、减计数，还是可逆计数。

如果没有把有效电平看清楚，后面的译码清零或置数状态很容易错一拍。

4.2 用计数器芯片构成任意进制计数器

常见方法有两类：清零法和置数法。

清零法

假设要构成 M 进制计数器，从 0 计到 M−1 ，当状态到达 M 时产生清零信号，把状态清为 0。

核心思路：

0,1,2,…,M−1→到达 M 后清零0

若清零端是异步清零，状态 M 一出现就被清掉，所以它通常只是一个很短的过渡状态，不属于稳定计数循环。

置数法

利用同步或异步置数端，把计数器装入指定初值。若从 N0 开始计到 N1 ，模数为：

M=N1−N0+1

如果是加计数器，可以让它在到达 N1 后重新置入 N0 。

清零法适合从 0 开始的模数变换；置数法适合需要从非零初值开始计数的题。

4.3 同步置数和异步清零的区别

这是芯片应用题最常错的点。

  • 异步清零：清零信号一有效，输出立即变为 0，不必等时钟。
  • 同步清零/同步置数：控制信号有效后，要等下一个有效时钟边沿才改变输出。

所以译码状态的选择不同：

  • 异步清零构成 M 进制时，通常译码状态 M 。
  • 同步置数构成 M 进制时，通常在状态 M−1 译码，使下一个时钟装入初值。

具体还要看题目给的芯片功能表。

4.4 级联计数器

当一个芯片的模数不够时，可以级联多个计数器。

若两个计数器模数分别为 M1 和 M2 ，常见级联总模数为：

M=M1M2

做级联题时先确定：

  1. 低位计数器何时产生进位或借位。
  2. 高位计数器是否由低位进位触发，还是由统一时钟加使能控制。
  3. 若要求任意模数，是否还需要整体译码清零或置数。

同步级联一般更稳定；异步级联结构简单，但高位状态会有传播延迟。

4.5 中规模移位寄存器应用

移位寄存器芯片常见功能：

  • 保持。
  • 左移。
  • 右移。
  • 并行置数。
  • 串行输入。
  • 串行输出。

典型应用：

应用 做法
串并转换 串行输入，若干个时钟后并行取出
并串转换 并行置数，再逐位移出
延时电路 数据经过 n 级移位后输出
环形计数器 末位反馈到首位
扭环形计数器 末位取反后反馈到首位
序列发生器 用反馈逻辑生成串行输入

环形计数器与扭环形计数器常用结论：

  • k 位环形计数器的模数通常为 k 。
  • k 位扭环形计数器的模数通常为 2k 。

这两类电路往往需要初始置数，否则可能进入全 0 或其他无效状态。

五、高频题型套路

5.1 触发器波形题

题目给 CP、输入端波形，让画 Q 。

套路：

  1. 标出 CP 的有效边沿或有效电平。
  2. 标出异步清零、置位有效区间。
  3. 在每个有效触发时刻代特性方程。
  4. 把 Q 的变化延后到触发时刻之后画出。

一句话：先看异步端，再看时钟，再看特性方程。

5.2 同步时序分析题

题目给逻辑图，让判断功能。

套路：

  1. 写 Ji,Ki 或 Di 。
  2. 写 Z 。
  3. 代入特性方程求 Qin+1 。
  4. 列状态表。
  5. 画状态图。
  6. 判断是几进制计数器、能否自启动、输出序列是什么。

5.3 异步计数器分析题

题目通常会给几个 T′ 或 JK 触发器级联。

套路：

  1. 看每一级触发器由谁触发。
  2. 从最低位开始分析每个有效边沿。
  3. 逐级判断有没有给下一级产生有效边沿。
  4. 写出状态序列。

不要把异步计数器当成同步计数器一次性更新。

5.4 计数器设计题

套路：

  1. 由模数确定触发器个数。
  2. 写状态循环。
  3. 列转换表。
  4. 根据 D 或 JK 激励表求输入方程。
  5. 画完整状态图检查自启动。

如果题目说“可自启动”，无效状态不能只写 X 后就结束。

5.5 芯片构成任意进制计数器

套路：

  1. 看芯片最大模数。
  2. 看清零/置数端有效电平。
  3. 决定用清零法还是置数法。
  4. 译码指定状态。
  5. 判断该状态是稳定有效状态还是瞬时过渡状态。

特别注意：异步清零译码 M ，同步置数常译码 M−1 ，但最终以芯片功能表为准。

六、易错点清单

  • RS 触发器的禁态取决于输入有效电平，不能只记 RS=0 。
  • D 触发器设计时最省事，直接令 Di=Qin+1 。
  • JK 触发器 J=K=1 是翻转。
  • T 触发器 T=1 翻转， T=0 保持。
  • T′ 触发器每来一个有效时钟就翻转，常用于二分频。
  • 边沿触发器只在有效边沿采样输入。
  • 主从 JK 触发器要注意 CP 有效期间输入变化造成的“一次翻转”问题。
  • 同步时序电路所有触发器同拍更新，异步时序电路要逐级传播分析。
  • 计数器模数等于有效循环状态数，不一定等于触发器所有可能状态数。
  • k 个触发器最多有 2k 个状态。
  • 自启动必须检查所有无效状态。
  • k 位环形计数器通常是 k 进制。
  • k 位扭环形计数器通常是 2k 进制。
  • k 位 m 序列长度为 2k−1 ，全 0 状态通常要单独处理。
  • 芯片应用题先看功能表，再决定译码清零或置数状态。

七、考前最小背诵版

如果时间很紧，至少记住下面这些。

触发器特性方程：

Qn+1=D Qn+1=JQn―+K―Qn Qn+1=Qn⊕T Qn+1=Qn―(T′ 触发器)

同步时序分析：

激励方程→状态方程→状态表→状态图→功能判断

计数器触发器个数：

M≤2k

无效状态数：

2k−M

JK 激励表：

转换 J K

0→1 1 X

1→1 X 0

芯片题关键判断：

异步清零：立即清零同步置数：等时钟装入
